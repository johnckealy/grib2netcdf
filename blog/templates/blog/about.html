Grib2netCDF

Grib2netCDF is a simple tool developed to convert GRIB files to netCDF format. It's completely free to use. GRIB (GRIdded Binary or General Regularly-distributed Information in Binary form) is a file format commonly used for meteorological data, and is a World Meteorological Organization (WMO) standard. Simply drag and drop a GRIB2 format file, and click Upload.

You can select your desired fields by ticking or unticking the checkboxes. When you're happy, click Create NetCDF file. Do bear in mind that netCDF files are inherently larger than GRIB, so try to select as few fields as possible from the checkbox lists; this will help our servers to keep up. If the site does crash (you'll get a Bad Gateway 502 error), don't worry about it, just try again with less fields selected. You can always break your requirements into multile netCDF files. We've also provided a sample GRIB2 file, taken from NOAA's NOMADs system, so you can try the tool out. You can select this file by clicking 'See a Sample' in the navbar.

That's it! It's a simple tool, intended to simplify the fields coming from the GRIB, allowing easier access to the data. That said, it is not fully comprehensive, and some of the fields in the GRIB file will not appear in the NetCDF. That's on purpose, the idea being to separate out the most useful fields.

Grib2netCDF was built and is maintained by John C. Kealy, a doctoral student of meteorology at the University of Exeter. Please send me an email with any comments, questions, or feature requests.
